# Telegram Chatbot

This is a Telegram chatbot developed using Node.js.

## Features
- Telegram bot integration
- NLP based responses
- Calculator functionality
- Firebase integration

## Technologies Used
- Node.js
- Telegram Bot API
- Firebase
- node-nlp

## How to Run
1. Install dependencies
   npm install
2. Create .env file
3. Add BOT_TOKEN in .env
4. Run the bot
   node bot.js
